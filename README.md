# Face Mask Detection

Easy to deploy face mask detection solution on any platform including linux, Windows, MacOS and even on Embedded Platform supporting Deep learning frameworks.

### Prerequisites

```
python ( >= 3.0)
Opencv ( >= 3.0)
tensorflow ( >= 2.0 )

```

## Running the tests

```
python3 face_mask.py

```
